# 枝理 Tab 隐私权政策

最后更新：2026 年 5 月 12 日

枝理 Tab（Tab Garden）是一个本地优先的浏览器标签页管理扩展。它的目标是帮助你在浏览器中整理、搜索和管理当前打开的标签页。

## 我们处理哪些数据

为了提供标签页管理功能，枝理 Tab 会在你的浏览器本地读取和处理以下信息：

- 当前打开的标签页标题、URL、域名和 favicon。
- 标签页所属窗口、是否为当前活跃标签、是否固定、是否正在播放声音、是否已休眠等浏览器状态。
- 用于识别重复标签、按域名分组、按窗口分组和执行关闭、切换、休眠等操作所需的轻量状态。
- 你在扩展内保存的设置，例如主题、语言、默认视图、强调色、动画开关、保护规则、白名单域名、搜索模式切换显示方式和诗文显示开关。

枝理 Tab 不读取网页正文内容，不注入 content script，也不会分析你正在浏览页面中的文本、表单或其他页面内容。

## 本地处理与存储

标签页整理、重复标签识别、搜索过滤、分组和设置管理均在你的浏览器本地完成。

枝理 Tab 使用 `chrome.storage.local` 保存扩展设置。在非扩展预览环境中，可能使用浏览器 `localStorage` 保存同类本地设置。以上数据不会被上传到枝理 Tab 开发者控制的服务器。

## 第三方服务和外部请求

枝理 Tab 当前可能在以下场景产生外部请求：

- **诗文展示**：当“展示诗文”功能开启时，扩展使用随安装包内置的本地诗文数据，并在本地按日期生成当天诗文列表，不会为此请求外部诗词接口。
- **网络搜索**：当你主动切换到网络搜索模式并提交关键词时，扩展会调用浏览器的默认搜索服务，在新标签页打开搜索结果。此时搜索关键词会发送给你当前浏览器配置的默认搜索引擎。
- **站点图标资源**：扩展优先使用浏览器标签页接口提供的 favicon，并在缺失时使用 Chrome/Edge 扩展内置的 favicon 能力读取浏览器本地可用的站点图标，不会为了获取 favicon 将标签页域名发送给枝理 Tab 开发者服务器。

除上述用户可见功能所需请求外，枝理 Tab 不会主动将你的标签页数据、URL、搜索记录或设置发送给开发者服务器。

## 数据共享与出售

枝理 Tab 不出售用户数据。

枝理 Tab 不会将你的标签页数据、URL、搜索记录或本地设置转让给第三方。只有在法律法规要求的情况下，才可能依法披露必要信息；但枝理 Tab 本身不运营用于收集这些数据的后端服务。

## 远程代码

枝理 Tab 不使用远程 JavaScript 或远程 WebAssembly 代码来执行扩展逻辑。扩展的核心代码随扩展包一同发布并在本地运行。

## 数据保留与删除

枝理 Tab 的设置保存在你的浏览器本地，保留时间取决于浏览器存储策略和你的操作。你可以通过以下方式删除这些数据：

- 在扩展设置中重置相关设置。
- 通过浏览器扩展管理页面移除枝理 Tab。
- 清理浏览器本地站点数据或扩展存储。

## 权限说明

枝理 Tab 使用以下浏览器权限：

- `tabs`：读取和管理当前打开的标签页，用于展示、搜索、切换、关闭和休眠标签。
- `windows`：读取窗口信息，用于按窗口分组标签页。
- `storage`：保存你的扩展设置。
- `search`：当你主动使用网络搜索模式时，调用浏览器默认搜索引擎打开搜索结果。
- 诗文展示使用内置本地数据，不需要额外站点访问权限。

枝理 Tab 不申请 `scripting`、`activeTab` 或 content script 权限。

## 联系方式

如果你对本隐私权政策或枝理 Tab 的数据处理方式有任何问题，请通过 Chrome Web Store / Edge Add-ons 商店页面或扩展发布者账号联系开发者。
