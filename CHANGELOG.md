# Changelog

## 0.1.6

> 何时一樽酒，重与细论文。

枝理 Tab 0.1.6 聚焦于分组卡片的空间展开体验：点击分组后，当前卡片会从原位置连续扩张为详情面板，内部标签从紧凑列表切换为完整条目，用户可以通过返回区域或 ESC 收起，让整理标签时的焦点切换更安静、更连贯。

### 新增

- 新增分组卡片 Shared Layout Focus Expand 交互，支持从概览网格连续展开到详情面板。
- 新增分组详情返回交互，可点击展开后的 header 区域或按 ESC 收起。

### 优化

- 优化分组展开时的布局承载方式，详情面板覆盖完整分组列表区域，避免非首个分组展开时出现顶部留白。
- 复用现有标签条目组件，在概览中保持紧凑模式，在详情中使用完整模式，不改变既有 UI 风格。

## 0.1.5

> 可怜楼上月徘徊，应照离人妆镜台。

枝理 Tab 0.1.5 是一次轻量补丁更新，主要补足了诗文区域的细节交互，并整理了应用内部状态结构，让新标签页在后续继续迭代时更稳、更容易维护。

### 新增

- 新增诗文单句复制操作，鼠标悬停在诗句行上即可快速复制当前这一句。

### 优化

- 优化诗文标题区域的悬浮操作呈现，复制全文、搜索诗文等入口更顺手。
- 优化部分搜索栏与设置面板的交互细节，让操作反馈更清晰自然。

### 稳定性

- 重构应用状态管理与标签结果渲染结构，减少主界面耦合，为后续功能迭代和稳定性修复打底。

## 0.1.4

> 江畔何人初见月，江月何年初照人？

枝理 Tab 0.1.4 聚焦于让新标签页更轻、更顺：诗文展开动画更柔和，悬浮操作可复制诗文或用浏览器默认搜索查询；搜索框聚焦态更清晰；应用结构拆分为更小的组件与 hooks，后续维护和迭代更稳定。同时移除自定义搜索引擎配置，改用浏览器默认搜索服务，并补充隐私说明。

### 新增

- 新增诗文复制与诗文搜索入口，可在诗文悬浮操作中快速使用。
- 新增隐私政策文档，说明权限、数据存储与第三方诗词服务使用方式。

### 优化

- 优化诗文展开、收起与浮起动画，让底部诗文呈现更自然。
- 优化搜索框聚焦态，提升键盘输入和视觉反馈的清晰度。
- 将主界面拆分为 Header、控制栏、标签区域、色彩菜单等组件，并抽取主题色相关 hooks。
- 精简样式表与点缀字体配置，移除冗余 CSS 规则。

### 调整

- 移除自定义搜索引擎选择，网络搜索统一使用浏览器默认搜索提供方。

## 0.1.3

> 庄生晓梦迷蝴蝶，望帝春心托杜鹃。

枝理 Tab 0.1.3 带来更安静也更实用的新标签页体验：新增底部诗文展示，可从今日诗词获取每日诗句，并支持点击展开完整内容；搜索栏升级为标签搜索与网络搜索双模式，可快速切换并选择 Google、DuckDuckGo、Bing 或 Baidu；同时优化了域名分组、特殊浏览器页面识别、设置面板和暗色主题细节，让标签整理、查找和浏览入口更加顺手。

### 新增

- 新增底部诗文展示，接入今日诗词 API，并支持点击展开诗文全文。
- 新增设置项，可控制是否展示底部诗文。
- 新增网络搜索模式，搜索框可在标签搜索和网络搜索之间切换。
- 新增搜索引擎选择，支持 Google、DuckDuckGo、Bing、Baidu。
- 新增搜索图标样式设置，可选择通用图标或当前搜索引擎图标。

### 优化

- 优化域名分组，支持识别 `google.com.hk`、`google.co.uk` 等多段公共后缀。
- 优化 `chrome://`、`edge://`、`about:blank`、扩展页等特殊页面的分组与展示。
- 优化设置面板、搜索模式切换、底部诗文、暗色主题和点缀字体等视觉细节。
- 集中整理基础字体栈与点缀字体变量，便于后续发布前检查字体授权、CSP、隐私和离线表现。

### 权限

- 新增 `https://v2.jinrishici.com/*` 站点权限，用于获取诗词内容。
