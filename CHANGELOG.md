# Changelog

## 0.1.3

> 庄生晓梦迷蝴蝶，望帝春心托杜鹃。

枝理 Tab 0.1.3 带来更安静也更实用的新标签页体验：新增底部诗文展示，可从今日诗词获取每日诗句，并支持点击展开完整内容；搜索栏升级为标签搜索与网络搜索双模式，可快速切换并选择 Google、DuckDuckGo、Bing 或 Baidu；同时优化了域名分组、特殊浏览器页面识别、设置面板和暗色主题细节，让标签整理、查找和浏览入口更加顺手。

### 新增

- 新增底部诗文展示，接入今日诗词 API，并支持点击展开诗文全文。
- 新增设置项，可控制是否展示底部诗文。
- 新增网络搜索模式，搜索框可在标签搜索和网络搜索之间切换。
- 新增搜索引擎选择，支持 Google、DuckDuckGo、Bing、Baidu。
- 新增搜索图标样式设置，可选择通用图标或当前搜索引擎图标。

### 优化

- 优化域名分组，支持识别 `google.com.hk`、`google.co.uk` 等多段公共后缀。
- 优化 `chrome://`、`edge://`、`about:blank`、扩展页等特殊页面的分组与展示。
- 优化设置面板、搜索模式切换、底部诗文、暗色主题和点缀字体等视觉细节。
- 集中整理基础字体栈与点缀字体变量，便于后续发布前检查字体授权、CSP、隐私和离线表现。

### 权限

- 新增 `https://v2.jinrishici.com/*` 站点权限，用于获取诗词内容。
